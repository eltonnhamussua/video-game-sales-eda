Visão Geral

Este projeto consiste numa Análise Exploratória de Dados (EDA) aplicada a uma base de dados sobre os videojogos mais vendidos da história, obtida no kaggle.
O objetivo principal é compreender padrões de vendas, identificar tendências relevantes e extrair insights que possam apoiar a tomada de decisões, servindo como primeira etapa de um pipeline completo de Análise de Dados, que será expandido futuramente.


Objetivos do Projeto

Explorar a estrutura geral do conjunto de dados
Identificar e tratar inconsistências, valores ausentes e problemas de formatação
Analisar padrões de vendas por:
    Plataforma
    Região
    Ano

Extrair insights iniciais que sirvam de base para análises mais avançadas no futuro


Fonte dos Dados
Base de dados obtida no kaggle
Dataset referente a vendas globais de videojogos
Dados utilizados exclusivamente para fins educacionais e analíticos


Etapas do Projeto

1. Visualização Inicial dos Dados
    Leitura do dataset
    Visualização do dataframe
    Identificação de colunas, tipos de dados e dimensões

2. Inspeção e Limpeza dos Dados
    Tratamento de valores ausentes
    Correção de inconsistências (ex.: erros de digitação, formatos diferentes)
    Padronização de variáveis categóricas
    Preparação do dataset para análise

3. Análise Exploratória de Dados (EDA)
    Estatísticas descritivas
    Análises por região, plataforma e ano
    Visualizações gráficas para apoio à interpretação
    Identificação de padrões e tendências relevantes


Tecnologias e Ferramentas Utilizadas
    Python — linguagem principal
    Pandas — manipulação e análise de dados
    Matplotlib — visualização de dados
    Seaborn — visualizações estatísticas avançadas
    Jupyter Notebook — ambiente de desenvolvimento

Próximos passos (planeados):
